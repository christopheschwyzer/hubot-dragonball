# Hubot-Dragonball

The Dragonball Image Generator.

## Installation

Add hubot-dragonball to your `package.json` dependencies.

```
"dependencies": {
  "hubot-dragonball": ">= 0.0.5"
}
```


Add hubot-dragonball to `external-scripts.json`
```
["hubot-dragonball", "more-scripts"]
```
